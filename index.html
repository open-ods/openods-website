The Project

OpenODS is an open project by @mattstibbs and @tonyyates using open NHS data.

It is changing regularly, therefore the consistency or accuracy of the data returned by the API cannot be guaranteed.

If have you have an suggestions or would like to help, please feel free to make a suggestion (lightbulb top right) or

Send email to info@openods.co.uk
